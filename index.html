To add more task pages

  1. Copy the template file in pages folder
    • And rename it to exercise_XX (replace x with exercise number)
  2. go to scripts -> nav.js
    • add the page name to the list at the top, this will generate the nav button on each page, so you wont have to add it everywhere
  3. in css folder, create the css file for that page
    1. In the html file, update the link so it points to the right css file

To get images from the exercises, just right click the image on the website and copy image link
